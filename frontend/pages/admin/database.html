Live Server Setup Guide
Step 1 — Upload Files

Upload the entire project to public_html/ (or a subdirectory) using cPanel File Manager or FTP.

Step 2 — Create the Database
  1. In cPanel, open MySQL Databases
  2. Create a new database (e.g. kapada_station)
  3. Create a database user and assign it All Privileges
  4. Open phpMyAdmin, select the new database
  5. Click Import and upload the .sql backup you downloaded above
Step 3 — Edit Backend Config

Edit backend/config/database.php with your hosting credentials:

define('DB_HOST',    'localhost');
define('DB_USER',    'cpanel_user_dbuser');
define('DB_PASS',    'your_strong_password');
define('DB_NAME',    'cpanel_user_kapada_station');
define('JWT_SECRET', 'change-this-to-a-long-random-string');
define('BASE_URL',   'https://yourdomain.com');
Step 4 — Set Upload Folder Permissions

In cPanel File Manager, set these folders to permission 755:

  • uploads/
  • uploads/products/
  • uploads/id_documents/
  • uploads/photos/
  • uploads/tryon/
Step 5 — Visit Your Site
  • Customer site: https://yourdomain.com/frontend/
  • Admin panel: https://yourdomain.com/frontend/pages/admin/dashboard.html
  • Default admin login: admin@kapadastationnew.com / Admin@123
Requirements

Ubuntu 20.04+ with Apache, PHP 7.4+, MySQL 5.7+ / MariaDB 10.3+

Install LAMP Stack
sudo apt update && sudo apt upgrade -y
sudo apt install apache2 php php-mysqli php-json php-fileinfo \
     mysql-server unzip -y
sudo a2enmod rewrite
sudo systemctl restart apache2
Deploy Files
# Copy project to web root
sudo cp -r kapadastationnew/ /var/www/html/kapada/
sudo chown -R www-data:www-data /var/www/html/kapada/
sudo chmod -R 755 /var/www/html/kapada/uploads/
Create Database
sudo mysql -u root -p
CREATE DATABASE kapada_station CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'kapada_user'@'localhost' IDENTIFIED BY 'StrongPass!123';
GRANT ALL PRIVILEGES ON kapada_station.* TO 'kapada_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;

# Import backup
mysql -u kapada_user -p kapada_station < kapada_station_backup.sql
Apache Virtual Host
<VirtualHost *:80>
    ServerName yourdomain.com
    DocumentRoot /var/www/html/kapada
    <Directory /var/www/html/kapada>
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
sudo a2ensite kapada.conf
sudo systemctl reload apache2
Set Environment Variables (recommended)

Add to /etc/apache2/envvars or your virtual host <VirtualHost> block:

SetEnv DB_HOST     localhost
SetEnv DB_USER     kapada_user
SetEnv DB_PASS     StrongPass!123
SetEnv DB_NAME     kapada_station
SetEnv JWT_SECRET  replace-with-64-char-random-string
SetEnv BASE_URL    https://yourdomain.com

Rather than editing backend/config/database.php directly, set these environment variables on your server. The config file reads them automatically.

Variable Example Value Description
DB_HOSTlocalhostMySQL host
DB_USERkapada_userMySQL username
DB_PASSStrongPass!123MySQL password
DB_NAMEkapada_stationDatabase name
JWT_SECRET64-char random stringJWT signing secret — never share this
BASE_URLhttps://yourdomain.comPublic URL of the site

Generate a secure JWT_SECRET with:
openssl rand -hex 32 (Linux/macOS) or use any online random string generator.
